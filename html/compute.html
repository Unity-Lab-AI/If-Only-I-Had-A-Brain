Unity GPU Compute Worker — donate your GPU to the brain

This tab donates your browser's WebGPU to Unity's brain. Each donor holds a full brain replica (data-parallel) — the server merges the Hebbian weight-deltas every donor computes and re-broadcasts them, so more donors mean more aggregate compute and redundancy. The brain auto-scales its neuron count to the connected community. The work is real neural matrix math — LIF spiking + sparse cross-projections, 100% equational, no text-AI / no language model. Leave the tab open to keep donating; one donor dropping off never downgrades the brain.

Donating overnight? This page now holds a screen Wake Lock and auto-recovers a lost GPU device — but for an unattended run also: (1) disable system sleep/hibernate in your OS power settings (a wake lock only blocks screen sleep), (2) keep this tab open and foreground, and (3) in chrome://discards set this tab to no auto-discard so Chrome doesn't unload it under memory pressure. A donor that stays connected uploads once and keeps training; a donor that drops forces a re-upload and stalls the brain.
